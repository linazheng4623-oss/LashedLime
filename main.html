Frequently Asked Questions

General Questions

Is Lime Client Safe?
Yes, this project is completely safe and open-source. You can review the code yourself to ensure there's no malicious content.
What Minecraft version is Lime Client for? 1.21.8
Lime Client is specifically designed for Minecraft 1.21.8. Make sure you're using this version to ensure compatibility and optimal performance.
Do I need Fabric API?
Yes, Lime Client requires Fabric API to function properly. You can download it from the link provided on the homepage or from the official Fabric website.
How do I install Lime Client?
  1. Download the latest version from the homepage
  2. Install Fabric Loader for Minecraft 1.21.8
  3. Download and install Fabric API
  4. Place the Lime Client JAR file in your mods folder
  5. Launch Minecraft with the Fabric profile

Technical Questions

Why isn't Lime Client working after installation?
Common issues include:
  • Using the wrong Minecraft version (must be 1.21.8)
  • Missing Fabric API installation
  • Outdated Java version
  • Conflicting mods in your mods folder
Try removing other mods first to check for conflicts.
Is Lime Client compatible with OptiFine?
Lime Client is built on Fabric, which generally doesn't support OptiFine. We recommend using Sodium and Iris Shaders instead for better performance and compatibility.
Can I use Lime Client on servers?
Lime Client works on most servers, but using certain features may violate server rules. Always check and respect individual server policies. Some anti-cheat systems may flag certain features.
How do I report bugs or suggest features?
You can report issues or suggest new features on our GitHub repository. Please provide detailed information about the bug or feature request.

Legal Questions

Is Lime Client allowed by Mojang's EULA?
Lime Client is a client modification, which is generally allowed by Mojang's EULA as long as it doesn't provide unfair advantages in multiplayer or enable piracy. However, server administrators may have their own rules regarding mod usage.
Is Lime Client free to use?
Yes, Lime Client is completely free and open-source. There are no premium features or paid versions.
